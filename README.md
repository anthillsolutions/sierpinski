Sierpinski Triangle

Install
-------

```
npm install
```

or

```
yarn
```

Contribution
------------

All contributions are welcomed. You can edit the code while seeing the change with this command:

```
$ ng serve
```

By default, it will connect to the server at the address `http://localhost:4200`.
